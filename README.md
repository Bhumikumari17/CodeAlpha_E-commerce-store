# 🍓 Berry Bloom - Full Stack E-Commerce Website

Berry Bloom is a responsive Full Stack E-Commerce Website developed using HTML, CSS, JavaScript, Node.js, Express.js, and MongoDB Atlas.

## 🌐 Live Demo

Frontend:
https://code-alpha-e-commerce-store-swart.vercel.app/

Backend API:
https://codealpha-e-commerce-store-he6e.onrender.com/

## 🚀 Features

- User Signup
- User Login
- Product Listing
- Add to Cart
- Shopping Cart
- Checkout Page
- Contact Page
- Responsive Design
- MongoDB Database Integration

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/bhumikumari17/CodeAlpha_E-commerce-Store.git
```

Go to project folder

```bash
cd CodeAlpha_E-commerce-Store
```

Install dependencies

```bash
npm install
```

Start backend

```bash
node backend/server.js
```

Open the frontend in your browser.
